## Clone the Repository
```sh
git clone https://github.com/Macahyba/proffy-mobile.git
```

## Install

```sh
cd proffy-mobile
```

## Usage

```sh
yarn start
```

> Follow `expo` instructions, by either running a mobile emulator or using the expo app (search for in your app store).
